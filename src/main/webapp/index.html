Intro.

JenSoft is is a pure java framework to make java charts components. API provides a complete charting solution for your Swing or Web Application. It provides advanced features you've come to expect with minimal integration efforts. Framework gives developers a very flexible toolkit solution for a wide variety of java chart and data visualization needs. The API can be used in many businesses such as reporting, health, finance or science and technology softwares.

Maven Dependency.

Core API

Core artifact 2.0.4 is available on central repository with the following dependency ids. Core artifact provides first-class comprehensive support for the newest chart developers 

<dependency>
    <groupId>org.jensoft</groupId>
    <artifactId>jensoft-core</artifactId>
    <version>2.0.4</version>
</dependency>
Sample Catalog.

Samples artifact 2.0.4 is available on central repository with the following dependency ids. Sample catalog provides first-class comprehensive chart views for each api feature. You can read this artifact online with the catalog sample browser. Browse whole catalog or by unit of chart type. 

<dependency>
    <groupId>org.jensoft</groupId>
    <artifactId>jensoft-samples</artifactId>
    <version>2.0.4</version>
</dependency>
